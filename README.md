# LikelihoodProfilerLab

[LikelihoodProfiler.jl](https://github.com/insysbio/LikelihoodProfiler.jl) experiments should go here.

## Benchmarks

- Which profiler is the best default for common model classes?
- Which methods are fastest for smooth, stiff, constrained, or poorly identifiable models?
- How robust are confidence interval endpoints across optimizers and profile strategies?

## Application to Optimal Experimental Design (OED)

- Experiments connecting profile likelihood confidence intervals with optimal experimental design criteria.
